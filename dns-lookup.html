What are DNS Records?

The Domain Name System (DNS) is the phonebook of the internet. It translates human-readable domain names (like google.com) into IP addresses (like 142.250.190.46) that computers use to talk to each other.

Common Record Types Explained

  • A Record: Points a domain to an IPv4 address (e.g., 192.168.1.1). This is the most common record for hosting websites.
  • AAAA Record: Points a domain to an IPv6 address. This is the modern successor to IPv4.
  • MX Record: Stands for "Mail Exchange". It tells the internet which mail server handles email for your domain.
  • TXT Record: Used for verification (like verifying your site with Google Console) and email security (SPF, DKIM).
  • NS Record: "Nameserver" records. They tell the internet which DNS provider manages your domain (e.g., Cloudflare, GoDaddy).

How this tool works

Unlike traditional command-line tools like `dig` or `nslookup`, this tool uses DNS-over-HTTPS (DoH) to securely fetch public records directly from Google's Public DNS servers. This ensures you see the most up-to-date information available globally.
