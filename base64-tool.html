What is Base64 Encoding?

Base64 is a binary-to-text encoding scheme that represents binary data (like images or executable files) in an ASCII string format. It is designed to ensure that data remains intact without modification during transport (e.g., via Email or HTTP APIs).

Why convert Images to Base64?

  • Reduce HTTP Requests: Instead of the browser fetching `logo.png` (1 request), you embed the Base64 string directly into the HTML `<img src="...">`. This speeds up page loading for small icons.
  • Email Signatures: Many email clients block external images by default. Embedding the image as Base64 ensures it always displays.
  • Database Storage: Storing small images directly in a text field in a database (though not always recommended for large files).
