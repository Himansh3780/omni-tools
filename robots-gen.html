What is a robots.txt file?

A robots.txt file is a simple text file that sits in the root directory of your website (e.g., www.yoursite.com/robots.txt). It provides instructions to "web robots" (like Googlebot) about which pages they are allowed to scan and index.

Why is it important for SEO?

  • Crawl Budget: Google doesn't have infinite resources. If you have thousands of useless pages (like admin panels or print versions of pages), you don't want Google wasting time crawling them.
  • Security: It keeps private directories (like /cgi-bin/ or /admin/) out of public search results.
  • Sitemap Discovery: It tells crawlers exactly where your Sitemap XML file is located.

Common Commands

User-agent: * (Applies to all robots)
Disallow: /admin/ (Do not look in the admin folder)
Allow: / (You are allowed to look everywhere else)
