What is URL Encoding?

URL Encoding (also known as Percent-Encoding) is a mechanism for encoding information in a Uniform Resource Identifier (URI). Characters that are not allowed in a URL (like spaces, emojis, or special symbols) must be converted into a valid ASCII format.

Why do spaces become %20?

The internet protocol does not allow spaces in URLs. Therefore, a space character is replaced by a percent sign followed by its hexadecimal ASCII code. For a space (ASCII 32), the hex value is 20, resulting in %20.

Common Encoded Characters

  • Space -> %20
  • @ (At Symbol) -> %40
  • / (Forward Slash) -> %2F
  • : (Colon) -> %3A
