What is a User Agent?

A User Agent (UA) is a string of text that your web browser sends to every website you visit. It acts like a digital ID card, telling the server which browser you are using, its version number, and your operating system.

Why do developers need this?

  • Debugging: If a user reports a bug, the developer needs to know if it's happening on Chrome, Safari, or Firefox to fix it.
  • Responsive Design: Websites use the UA string to decide whether to serve the "Mobile" version or the "Desktop" version of a page.
  • Bot Detection: Servers check the UA to identify if a visitor is a human or a crawler (like Googlebot).

Common User Agent Tokens

You might see strange words like "Mozilla/5.0" or "AppleWebKit" in your string even if you use Chrome. This is for historical compatibility reasons. Modern browsers include these tokens so that older websites render correctly.
