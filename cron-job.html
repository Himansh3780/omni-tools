What is a Cron Job?

Cron is a time-based job scheduler in Unix-like computer operating systems (like Linux and macOS). Developers use cron to schedule jobs (commands or scripts) to run periodically at fixed times, dates, or intervals.

Understanding Crontab Syntax

A standard cron expression consists of five fields separated by whitespace:

* * * * * command_to_execute
| | | | |
| | | | +----- Day of Week (0 - 7) (Sunday=0 or 7)
| | | +------- Month (1 - 12)
| | +--------- Day of Month (1 - 31)
| +----------- Hour (0 - 23)
+------------- Minute (0 - 59)

Common Cron Examples

  • 0 0 * * * - Run once a day at midnight.
  • */5 * * * * - Run every 5 minutes.
  • 0 0 1 * * - Run on the first day of every month.
  • 0 0 * * 5 - Run every Friday at midnight.
