What is Hashing?

Hashing is the process of converting data (text, numbers, or files) into a fixed-length string of characters. Unlike encryption, hashing is a one-way functionâ€”you cannot retrieve the original data from the hash.

Why use SHA-256?

  • Password Security: Websites store the hash of your password, not the password itself. If their database is leaked, hackers only see random characters.
  • File Integrity: When you download a software file, the developer provides a "Checksum" (Hash). You can hash your downloaded file and compare it. If they match, the file is safe and hasn't been tampered with.

Supported Algorithms

SHA-256 is the current industry standard used by Bitcoin and SSL certificates. SHA-512 provides even higher security. SHA-1 is older and considered less secure, but still used for verifying old files.
