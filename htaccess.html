What is an .htaccess file?

An .htaccess (Hypertext Access) file is a configuration file used on web servers running the Apache software. It allows you to modify server settings for specific directories without needing access to the main server configuration file.

Why use .htaccess?

  • 301 Redirects: Permanently move a URL from one place to another. Essential for maintaining SEO when you change your site structure.
  • Security: You can password protect folders, block specific IP addresses, or disable directory listings so hackers can't see your files.
  • Performance: You can enable GZIP compression and browser caching to make your site load faster.

Warning

The .htaccess file is extremely powerful. A single syntax error (like a missing space) can cause your entire website to crash with a "500 Internal Server Error." Always back up your existing file before uploading a new one.
